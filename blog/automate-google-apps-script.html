Automate Your Admin Work with Google Apps Script and Claude

Gab · March 2026 · 6 min read

If you run a small business or a side project, you probably spend too much time on "admin glue work" — moving data from an email to a spreadsheet, saving attachments to Drive, or sending calendar invites.

Google has a built-in tool called Google Apps Script that can automate almost anything across Gmail, Sheets, Docs, and Drive. It's powerful, but it requires writing JavaScript.

Here's the secret: You don't need to know JavaScript. You just need to know how to ask Claude to write it for you.

What is Google Apps Script?

Think of it as the "macro" language for the web. It runs on Google's servers, not your computer. You can set triggers so scripts run automatically (e.g., every hour, or whenever a form is submitted).

Example 1: Save Email Invoices to Drive

Let's say you get invoices from various SaaS tools (GitHub, Vercel, OpenAI) in your inbox. You want them saved to a "Receipts/March 2026" folder in Google Drive.

The Prompt for Claude:

Write a Google Apps Script that searches my Gmail for threads with the label "Invoices".
For each email found:
1. Check if it has a PDF attachment.
2. If yes, save the attachment to a Google Drive folder named "Receipts/[Current Month Year]".
3. If the folder doesn't exist, create it.
4. Mark the email as read and remove the "Invoices" label.

Claude will generate a function like processInvoices(). You copy-paste it into the script editor (extensions > Apps Script in Sheets, or script.google.com), save, and run.

Example 2: Log Contact Form Submissions to Sheets

If you receive lead notifications via email (e.g., from a WordPress site or a simple HTML form), you might want them in a spreadsheet to track follow-ups.

The Prompt for Claude:

I receive emails with the subject "New Lead".
The body looks like:
Name: John Doe
Email: john@example.com
Message: Interested in services.

Write a script to:
1. Find all unread emails with subject "New Lead".
2. Parse the body to extract Name, Email, and Message.
3. Append a new row to the active Google Sheet with [Date, Name, Email, Message].
4. Mark the email as read.

Now your spreadsheet fills itself up automatically.

How to Set It Up

  1. Go to script.google.com or open a Google Sheet and click Extensions > Apps Script.
  2. Delete the default myFunction code.
  3. Paste the code Claude gave you.
  4. Click the "Save" icon.
  5. Click "Run" to test it. Google will ask for permissions — grant them.
  6. To make it run automatically, click the Clock icon (Triggers) on the left sidebar.
    • Choose your function.
    • Select event source: Time-driven.
    • Select type: Hourly timer (or daily, depending on your needs).

Why This is Better Than Zapier

Tools like Zapier or Make are great, but they get expensive once you start processing high volumes. Google Apps Script is free (within generous quotas). For simple tasks involving Google's own ecosystem, it's often faster and more reliable because it's native.

Plus, with Claude writing the code, the technical barrier is gone. You describe the workflow; Claude handles the syntax.
